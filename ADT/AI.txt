*12 explain minmax procedure with example
-> it is a decision making algo used in game theory to minimize the maximum potential loss
   is a decision making algo used in game theory and AI for two player such as chess,tic tac toe 
   the procedure works recursively
   max player - tries to maximize their score
   min player - tries to minimize the max players score
   procedure 
   1. generate the entire game tree down to the terminal state(game over)
   2. assign scores to the terminal states
   3. back the scores up the tree
   4. select the moves at the current state
   eg.tic tac toe
   in tic tac toe ,terminal state are evaluated as +1 for a win,-1 for a loss,and 0 for a draw
   the max player chooses moves that lead nodes with higher values 
   while the min player chooses moves leading to lower values
   the algo used for the best possible outcome

*22 what is hill climbing write algorithm for it
-> a simple optimization algo that iteratively moves to the best neighboring solution
   is used for optimization problems
   it is informed search
   it is greedy approach
   it starts with initial state and iteratively moves to a neighbouring state with a better value
   algo
   1.initialize the current state as the starting point
   2.loop - 
      1.evaluate all neighboring states of the current state.
      2.select the neighbour that provides the best value
      3.if the selected neighbour is better than the current state, move to that neighbour and set it as the new current state
      4.else(no neighbour is better) stop the loop.the current state is a local optimum
